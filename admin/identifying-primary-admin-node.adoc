= 确定主管理节点
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
主管理节点托管 CMN 服务。某些维护过程只能使用主管理节点执行。

.您需要的内容
* 您必须使用支持的浏览器登录到网格管理器。
* 您必须具有特定的访问权限。


.步骤
. 选择*支持*>*工具*>*网格拓扑*。
. 选择*；site_****管理节点*、然后单击 image:../media/icon_plus_sign_black_on_white.gif["加号"] 展开拓扑树并显示此管理节点上托管的服务。
+
主管理节点托管 CMN 服务。

. 如果此管理节点不托管 CMN 服务，请检查其他管理节点。
