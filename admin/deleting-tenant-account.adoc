= 删除租户帐户
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果要永久删除租户对系统的访问权限，可以删除租户帐户。

.您需要的内容
* 您必须使用支持的浏览器登录到网格管理器。
* 您必须具有特定的访问权限。
* 您必须已删除与租户帐户关联的所有分段（ S3 ），容器（ Swift ）和对象。


.步骤
. 选择*租户*。
. 选择要删除的租户帐户。
+
如果您的系统包含20个以上的项目、则可以指定一次在每个页面上显示的行数。使用搜索框按显示名称或租户ID搜索租户帐户。

. 从*操作*下拉列表中、选择*删除*。
. 选择 * 确定 * 。
