= 将自定义事件计数重置为零
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果只想重置自定义事件的计数器，则必须使用支持菜单中的网格拓扑页面。

.关于此任务
重置计数器会导致下一个事件触发警报。相反，确认警报时，只有在达到下一阈值级别时才会重新触发该警报。

. 选择*支持*>*工具*>*网格拓扑*。
. 选择 *_GRID NODE_* > * SSM* > * 事件 * > * 配置 * > * 主 * 。
. 选中自定义事件的 * 重置 * 复选框。
+
image::../media/custom_events_reset.gif[屏幕截图，位于 "SMS">" 事件 ">" 配置 ">" 主 "]

. 单击 * 应用更改 * 。
