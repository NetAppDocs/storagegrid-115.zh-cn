= 活动，空闲和并发 HTTP 连接的优势
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如何配置 HTTP 连接可能会影响 StorageGRID 系统的性能。根据 HTTP 连接是活动连接还是空闲连接，或者您有多个并发连接，配置会有所不同。

您可以确定以下类型的 HTTP 连接的性能优势：

* 空闲 HTTP 连接
* 活动 HTTP 连接
* 并发 HTTP 连接
